# Security Policy

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead, report them via [GitHub's private vulnerability reporting](https://github.com/slashgear/team-bookmarks/security/advisories/new) or by emailing the maintainer directly.

Include as much detail as possible: steps to reproduce, potential impact, and any suggested fix.

We will acknowledge your report within 48 hours and aim to release a fix within 7 days for critical issues.
